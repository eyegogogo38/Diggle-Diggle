# Diggle Diggle 🐹 디글디글

귀여운 두더지 **디굴이**와 함께하는 땅 속에 갇힌 모험! 
살쾡이와 멧돼지를 피해 자원을 모으고 살아남으세요.

## ✨ 특징
- **귀여운 그래픽:** 카카오 '더찌' 스타일의 2등신 두더지 캐릭터.
- **PWA 지원:** PC 및 모바일에서 앱으로 설치하여 언제 어디서나 플레이 가능.
- **다양한 아이템:** 도토리(탄약), 금화, 버섯, 뿌리 등 땅 속 자원 수집.
- **성장 시스템:** 스테이지가 올라갈수록 적들이 강력해지며 긴장감 상승.

## 🎮 플레이 방법
1. 브라우저에서 `index.html`을 엽니다.
2. 이름을 입력하고 게임을 시작합니다.
3. **이동:** 방향키 (Up, Down, Left, Right)
4. **공격:** Space (도토리 발사)
5. **디깅:** Z (땅 파기 / 터널 생성)

## 🛠 기술 스택
- Vanilla JavaScript (Canvas API)
- HTML5 / CSS3
- PWA (Service Worker, Web Manifest)
